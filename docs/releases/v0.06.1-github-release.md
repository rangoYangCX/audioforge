# AudioForge 0.06.1 - Workspace Simplification and Smoke Baseline Refresh

AudioForge 0.06.1 聚焦的是桌面工具工作台的继续收口，以及把仓库里的 smoke 基线和交付文档刷新到当前可验证状态。这个版本不改 Unity 运行时契约，不要求 Unity 项目改 SDK 接口或导出消费方式。

## Highlights

- 欢迎页进一步收敛为任务型首页，保留新建工程、打开工程以及直达事件设计、资源整理、Bus 混音台和结果中心的入口。
- 底部结果坞改为默认紧凑、按需展开；完整回看仍统一进入结果中心，减少常驻大面板占位。
- 工作区 overview、note card 与状态条继续去重，重复说明和重复状态回收到已有工作区上下文和结果摘要里。
- 主文档、空项目验证说明、内部上线执行表与 smoke 报告已刷新到当前基线。

## Compatibility

- 0.06.1 的主要变化发生在桌面工具侧的工作台信息组织、结果坞表现和交付文档上。
- Unity 导出契约、资源目录结构和 `unity_package/Assets/AudioForgeRuntime` 下的运行时代码保持不变。
- 如果 Unity 同学已经在对接旧包，可以继续沿用；只有需要同步新版文档、副本验证材料或重新导出的样例时，才需要重新生成 `AudioForgeUnityPackage-0.06.1`。
