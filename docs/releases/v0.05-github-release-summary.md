# AudioForge 0.05 Summary

AudioForge 0.05 聚焦大工程下的构建效率和可追踪性。桌面工具现已支持全量构建、增量构建和选中构建三种范围，并在不改变 Unity 现有接入主契约的前提下，引入资源级复用、构建计划预览和版本化变更记录。

## Highlights

- 新增全量构建、增量构建、选中构建三种范围，减少大工程下不必要的全量重建。
- 构建交付页新增构建计划预览，可直接看到请求范围、实际执行范围、重建资源数、复用资源数和移除资源数。
- 导出器现会全量刷新元数据，同时复用未变化音频资源，只重建脏资源。
- `AudioManifest.json` 新增 `BuildFingerprint`，`BuildReport.json` 新增 `BuildPlan`，方便联调、审计和 CI 跟踪差异。
- 选中构建若发现选区外脏资源，会自动升级为增量构建，优先保证完整包一致性。

## Unity Impact

- Unity 现有接入主契约不变。
- 新增字段主要用于构建审计和差异追踪，不要求现有运行时代码立即改造。
