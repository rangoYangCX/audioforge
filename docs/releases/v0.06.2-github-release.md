# AudioForge 0.06.2 - Recent Preview Card Polish

AudioForge 0.06.2 聚焦的是桌面工具右侧最近试听卡片的继续收口，让全局试听会话在跨工作区存在时更省空间、更清晰，也更接近一个真正的 transport card。这个版本不改 Unity 运行时契约，不要求 Unity 项目改 SDK 接口或导出消费方式。

## Highlights

- 最近试听侧栏卡片收口为更明确的 transport card，标题、状态、控制按钮和响度摘要分层展示。
- 最近试听按钮继续放大，并补上播放中、暂停、可试听和可重播状态徽标，跨流程控制更直观。
- 最近试听卡片与响度监视器的摘要标签已拆开，侧栏卡片不会再因后续视图挂载而丢失内部指标内容。

## Compatibility

- 0.06.2 的主要变化全部发生在桌面工具视图层和交付文档层。
- Unity 导出契约、资源目录结构和 `unity_package/Assets/AudioForgeRuntime` 下的运行时代码保持不变。
- 如果 Unity 同学已经在对接旧包，可以继续沿用；只有需要同步新版文档、副本验证材料或重新导出的样例时，才需要重新生成 `AudioForgeUnityPackage-0.06.2`。

## Validation

- PySide 离屏 recent preview 回归断言通过：侧栏宽度、按钮尺寸、状态徽标、暂停/继续信号切换和卡片内指标挂载均符合预期。
- `audioforge/app/views/main_window.py` 与 `tests/unit/test_main_controller_layout.py` 静态错误检查通过。
